## 系统架构

### 三层混合架构

```
┌─────────────────────────────────────────────────────┐
│                    用户输入                           │
└──────────────────┬──────────────────────────────────┘
                   │
                   ▼
        ┌──────────────────────┐
        │   路由分类层 (TextCNN)  │
        │   - 延迟: <1ms        │
        │   - 准确率: 99.51%    │
        │   - 参数: 187K        │
        └──────────┬───────────┘
                   │
                   ▼
        ┌──────────────────────┐
        │   决策引擎            │
        │   - 直接执行          │
        │   - 技能检索          │
        │   - LLM回退          │
        └──────────┬───────────┘
                   │
        ┌──────────┴──────────┐
        ▼                     ▼
┌──────────────┐    ┌──────────────┐
│  技能检索层   │    │  多轮对话层   │
│  - 574技能   │    │  - 上下文理解 │
│  - <10ms延迟  │    │  - 智能追问   │
└──────┬───────┘    └──────────────┘
       │
       ▼
┌──────────────────────┐
│   LLM解析层 (可选)    │
│   - 复杂意图理解     │
│   - 参数提取         │
│   - 延迟: <300ms     │
└──────────┬───────────┘
           │
           ▼
┌──────────────────────┐
│   执行引擎            │
│   - 技能调用         │
│   - 参数传递         │
└──────────────────────┘
```

### 决策逻辑

| 置信度 | 输入类型 | 处理路径 | 说明 |
|--------|----------|----------|------|
| >0.9 | 简单指令 | 直接执行 | 路由高置信度+简单表达 |
| 0.7-0.9 | 任意 | 技能检索 | 路由中高置信度 |
| <0.7 | 任意 | LLM回退 | 路由低置信度 |
| 任意 | 模糊 | 多轮对话 | 需要追问 |

## 使用示例

### Python API

```python
from src.hybrid.inference_engine import InferenceEngine

# 初始化引擎
engine = InferenceEngine(
    router_model_path="data/models/router_clean_final/best_model.pth",
    skills_dir="vc/skills",
    use_llm=False,
    device="cuda"
)

# 处理用户输入
result = engine.process("打开空调")

print(f"类别: {result.category}")
print(f"技能: {result.skill_name}")
print(f"参数: {result.parameters}")
print(f"置信度: {result.confidence}")
print(f"延迟: {result.latency_ms}ms")
```
